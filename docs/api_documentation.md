# API Documentation

## Overview
This project integrates with two main data sources:
- **Alpha Vantage API**: Financial statement data
- **Yahoo Finance (via yfinance)**: Historical stock data

## Configuration
API keys are managed through environment variables. See `config.yaml` for endpoint configurations.

## Rate Limits
- Alpha Vantage: 5 API requests per minute
- Yahoo Finance: No strict limits, but be respectful
